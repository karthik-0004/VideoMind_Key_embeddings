# Video-Knowledge-Extraction-Semantic-Search-System-RAG-based-
An end-to-end pipeline to convert long videos into searchable knowledge using speech-to-text and embeddings.– Implemented semantic search to retrieve relevant answers and generate timestamped, structured study PDFs using local LLMs.
### Create a folder called videos and pre-download all required videos into it before executing the program.
### U need a Groq API key make sure u have it , enclose it in .env file and use it
